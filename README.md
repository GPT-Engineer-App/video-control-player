# video-control-player

Create a JavaScript video player that can play videos forward, backward, and pause at different timestamps, with click-targets overlaid on the video to control these actions.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/video-control-player.git
cd video-control-player
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
